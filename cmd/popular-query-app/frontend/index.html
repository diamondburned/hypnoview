Popular Query Generator

Note: Larger time periods will take longer to load. Be patient!

This is a simple tool that generates query strings for the Hypnohub image board. It's meant to replicate the previous Hypnohub's popular queries feature, which is now removed after the site's redesign.

To use this tool, simply click on one of the buttons above. The query string will be generated for you. After that, just click the spiraly button!
